## 2️⃣ What is the spread operator (...)?
 `spread` operator is a ES6 featurs. it spreads the element of a iterable (array, string, Map) or the properties of an object. it is used for
Copying arrays, Merging arrays, Finding max value using Math.max(...arr) etc.

arrow function is a more concise way to write function expressions using the `=>` syntax.it is a es6 feature.
